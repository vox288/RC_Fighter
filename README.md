<h2>RC_Fighter is a short top-down shooter game</h2> 
                    
- It was a little side project of mine to dive a little deeper 
into OOP.
                  
<img width="640" height="480" alt="Screenshot 2026-03-12 215659" src="https://github.com/user-attachments/assets/2002479a-9553-454d-8cb6-580465b5874e" /> 
                  
- Learning more about some OOP techniques and creating simple sprite-animations and soundeffects was a lot of fun
- Sprites were created with <a href="https://libresprite.github.io/#!/">LibreSprite</a>
- soundeffects were created with <a href="https://www.audacityteam.org">Audacity</a>
- The game was written using <a href="https://www.pygame.org/news">PyGame</a>
- pygwidgets and pyghelpers by <a href="https://github.com/IrvKalb/">Irv Kalb</a> and his book Object-Oriented Python sure helped a lot
